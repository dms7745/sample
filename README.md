# 📖 입트영(입이 트이는 영어) 스마트 학습 플랫폼 (LMS)
본 프로젝트는 학습 효율을 극대화하기 위해 설계된 맞춤형 교육 관리 시스템입니다.
수강생의 학습 여정을 데이터로 관리하여 체계적인 영어 습득 환경을 제공합니다.

https://lms.da2un.store
!!!!!!!
####
🎯 기획 및 제작 동기
학습 환경의 통합: 텍스트와 오디오가 분리된 기존 학습 방식의 불편함을 해소하고자 통합 인터페이스를 기획했습니다.

데이터 기반 관리: 사용자가 자신의 학습 진도와 성취도를 직관적으로 파악할 수 있는 대시보드 시스템의 필요성을 느껴 개발을 시작했습니다.

기술적 도전: Spring Boot와 JPA를 활용하여 복잡한 회원-강의-학습 이력 간의 관계를 설계하고 배포하는 전체 개발 사이클을 경험하고자 했습니다.

🛠 주요 기술 스택 (Tech Stack)
Backend

Java 17 / Spring Boot 3.x

Spring Data JPA: 데이터 접근 계층 추상화 및 유지보수성 향상

MySQL: 관계형 데이터베이스를 통한 안정적인 데이터 관리

Frontend

Thymeleaf: 서버 사이드 렌더링을 통한 동적 콘텐츠 제공

Bootstrap 5: 모든 디바이스에 최적화된 반응형 UI 구현

JavaScript (ES6+): 대화형 사용자 경험 제공

Infrastructure

AWS EC2 / Ubuntu 22.04 LTS

Nginx: 리버스 프록시 및 웹 서버 운용

🚀 핵심 구현 기능
지능형 대시보드: 개인별 학습 진도율 시각화 및 최근 학습 내역 자동 연결

커리큘럼 관리: 교재 Unit별 강의 콘텐츠 구성 및 단계별 학습 환경 제공

회원 맞춤형 시스템: 마이페이지를 통한 개인 정보 및 학습 이력 통합 관리

커뮤니티: 학습자 간의 질문과 답변을 위한 인터랙티브 게시판 운영

📐 시스템 설계 및 아키텍처
DB 모델링: 정규화를 통해 데이터 중복을 최소화하고, 확장성을 고려한 ERD 설계

보안: Spring Security(또는 세션 방식)를 활용한 사용자 인증 및 인가 처리

📈 프로젝트를 통해 얻은 인사이트
클라이언트의 요구사항을 데이터베이스 스키마로 변환하는 설계 역량을 강화했습니다.

AWS 환경에서 Nginx를 설정하고 실제 서비스를 배포하며 인프라 운영에 대한 이해도를 높였습니다.
